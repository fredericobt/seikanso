This site is built with [Astro](https://astro.build) and uses the Kanso theme — a minimal, typographic design focused on reading and writing.

## The Author

Frederico BT is a developer and writer based in Brazil.

## The Stack

- **Framework:** Astro
- **Styling:** Pure CSS with custom properties
- **Fonts:** Inter (self-hosted)
- **CMS:** Decap CMS
- **Search:** Pagefind
- **Comments:** Webmentions
